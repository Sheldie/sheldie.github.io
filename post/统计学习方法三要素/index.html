统计学习方法三要素

2020-09-11 / 15 min read

统计学习方法 = 模型 + 策略 + 算法

监督学习问题就是 经验风险 或 结构风险 函数的 最优化问题，这时经验或结构风险函数是最优化的目标函数。

原文内容





Key points

#1：参数空间可以定义为条件概率的集合。

由决策函数表示的模型为非概率模型，由条件概率表示的模型为概率模型。

#2：根据大数定律，当样本容量N趋于无穷时，经验风险趋于期望风险。

n充分大时（趋于无穷大），

  • 伯努利大数定律：n次伯努利实验 → 事件发生的频率趋于概率
  • 切比雪夫大数定律：期望和方差存在且一致有界 → 算数平均值趋于数学期望
  • 辛钦大数定律：独立同分布 → 样本均值依概率收敛于期望值
#3：当模型是条件概率分布、损失函数是对数损失函数时，经验风险最小化就等价于极大似然估计。

​ 对于这个函数：P(x∣θ)P(x|\theta)P(x∣θ)

​ 输入有两个： xxx 表示某一个具体的数据； θ\thetaθ 表示模型的参数。

  • 如果 θ\thetaθ 是已知确定的，xxx 是变量，这个函数叫概率函数（probability function），它描述对于不同的样本点 xxx ，其出现概率是多少。
  • 如果 xxx 是已知确定的， θ\thetaθ 是变量，这个函数叫似然函数（likelihood function）, 它描述对于不同的模型参数，出现x这个样本点的概率是多少。

极大似然估计（maximum likelihood estimation）的原理：（频率学派）

给定一个概率分布 DDD，已知其概率密度函数（连续分布）或概率质量函数（离散分布）为 fDf_DfD​，以及一个分布参数 θ\thetaθ，我们可以从这个分布中抽出一个具有 nnn 个值的采样 X1,X2,…,XnX_1, X_2,\ldots, X_nX1​,X2​,…,Xn​，利用 fDf_DfD​ 计算出其似然函数：

L(θ∣x1,…,xn)=fθ(x1,…,xn).L(\theta \mid x_{1},\dots ,x_{n})=f_{\theta }(x_{1},\dots ,x_{n}). L(θ∣x1​,…,xn​)=fθ​(x1​,…,xn​).

若 DDD 是离散分布，θ\thetaθ 即是在参数为 θ\thetaθ 时观测到这一采样的概率。若其是连续分布，θ\thetaθ 则为 X1,X2,…,XnX_1, X_2,\ldots, X_nX1​,X2​,…,Xn​ 联合分布的概率密度函数在观测值处的取值。一旦我们获得 X1,X2,…,XnX_1, X_2,\ldots, X_nX1​,X2​,…,Xn​ ，我们就能求得一个关于 θ\thetaθ 的估计。最大似然估计会寻找关于 θ\thetaθ 的最可能的值（即，在所有可能的 θ\thetaθ 取值中，寻找一个值使这个采样的“可能性”最大化）。从数学上来说，我们可以在 θ\thetaθ 的所有可能取值中寻找一个值使得似然函数取到最大值。这个使可能性最大的 θ^\widehat{\theta}θ 值即称为 θ\thetaθ 的最大似然估计。由定义，最大似然估计是样本的函数。

若总体 XXX 为连续型，其概率密度函数为 f(x;θ)f(x;\theta)f(x;θ) ，其中 θ\thetaθ 为未知参数。设 X1,X2,…,XnX_1, X_2,\ldots, X_nX1​,X2​,…,Xn​ 是取自总体的样本容量为n的简单样本，则 X1,X2,…,XnX_1, X_2,\ldots, X_nX1​,X2​,…,Xn​ 的联合概率密度函数为 ∏i=1nf(xi;θ)\prod_{i=1}^n f(x_i;\theta)∏i=1n​f(xi​;θ) 。

又设 X1,X2,…,XnX_1, X_2,\ldots, X_nX1​,X2​,…,Xn​ 的一组观测值为 x1,x2,…,xnx_1, x_2,\ldots, x_nx1​,x2​,…,xn​，则随机点 X1,X2,…,XnX_1, X_2,\ldots, X_nX1​,X2​,…,Xn​ 落在点 x1,x2,…,xnx_1, x_2,\ldots, x_nx1​,x2​,…,xn​ 的邻边（边长分别为 dx1,dx2,…,dxndx_1, dx_2,\ldots, dx_ndx1​,dx2​,…,dxn​ 的n维立方体）内的概率近似地为 ∏i=1nf(xi;θ)dxi\prod_{i=1}^n f(x_i;\theta)dx_i∏i=1n​f(xi​;θ)dxi​。

L(θ)=L(x1,x2…,xn;θ)=∏i=1nf(xi;θ)L(\theta)=L(x_{1}, x_2\dots ,x_{n};\theta)= \prod_{i=1}^n f(x_i;\theta) L(θ)=L(x1​,x2​…,xn​;θ)=i=1∏n​f(xi​;θ)

L(θ)L(\theta)L(θ)称为样本的似然函数。 极大似然估计法原理就是固定样本观测值 ，挑选参数 θ\thetaθ 使

L(x1,x2…,xn;θ^)=maxL(x1,x2…,xn;θ)L(x_{1}, x_2\dots ,x_{n};\widehat\theta)=maxL(x_{1}, x_2\dots ,x_{n};\theta) L(x1​,x2​…,xn​;θ)=maxL(x1​,x2​…,xn​;θ)

这样得到的 θ^\widehat\thetaθ 与样本值有关，θ^(x1,x2,…,xn)\widehat\theta(x_1, x_2, \ldots, x_n)θ(x1​,x2​,…,xn​) 称为参数 θ\thetaθ 的极大似然估计值，其相应的统计量 θ^(X1,X2,…,Xn)\widehat\theta(X_1, X_2, \ldots, X_n)θ(X1​,X2​,…,Xn​)称为 θ\thetaθ 的极大似然估计量。

求极大似然估计 θ^\widehat\thetaθ，更多情况下是利用 lnL(θ)lnL(\theta)lnL(θ) 是 L(θ)L(\theta)L(θ) 的增函数，故 lnL(θ)lnL(\theta)lnL(θ) 与 L(θ)L(\theta)L(θ) 在同一点处达到最大值，于是对似然函数 L(θ)L(\theta)L(θ) 取对数，利用微分学知识转化为求解对数似然方程：

lnL(θ)=∑i=1nlnf(xi∣θ)lnL(\theta)=\sum_{i=1}^n lnf(x_i \mid \theta) lnL(θ)=i=1∑n​lnf(xi​∣θ)

对比 对数损失函数（logarithmic loss function）或对数似然损失函数（log-likelihood loss function）

L(Y,P(Y∣X))=−log⁡P(Y∣X)L(Y,P(Y \mid X)) = -\log P(Y \mid X) L(Y,P(Y∣X))=−logP(Y∣X)

经验风险最小化（empirical risk minimization，ERM）：

min⁡f∈F1N∑i=1NL(yi,f(xi))\min_{f\in\mathcal{F}} \frac{1}{N} \sum_{i=1}^N L(y_i,f(x_i)) f∈Fmin​N1​i=1∑N​L(yi​,f(xi​))

其中 F\mathcal{F}F 为假设空间，可以看出当模型是条件概率分布、损失函数是对数损失函数时，经验风险最小化就等价于极大似然估计。

#4：当模型是条件概率分布、损失函数是对数损失函数、模型复杂度由模型的先验概率表示时，结构风险最小化就等价于最大后验概率估计。

最大后验估计（maximum a posteriori estimation）的原理：（贝叶斯派）

极大似然估计其实就是求一组能够使似然函数最大的参数，即

θ^ML(x)=arg⁡max⁡θf(x∣θ)\widehatθ_{ML}(x)= \arg \max_θf(x \mid θ) θML​(x)=argθmax​f(x∣θ)

此时最大后验估计的思想是：假设 θ\thetaθ 也有先验概率，则有贝叶斯公式

p(θ∣x)=p(x∣θ)×p(θ)P(x)p(\theta \mid x)=\frac{p(x \mid \theta) \times p(\theta)}{P(x)} p(θ∣x)=P(x)p(x∣θ)×p(θ)​

其中 p(x∣θ)p(x \mid \theta)p(x∣θ) 就是似然函数，p(θ)p(\theta)p(θ) 是先验概率，是指在没有任何实验数据的时候对参数 θ\thetaθ 的经验判断。

最大后验估计优化的就是后验概率，即给定观测值后使后验概率最大：

θ^MAP=arg⁡max⁡θp(θ∣x)=arg⁡max⁡θp(x∣θ)×p(θ)P(x)=arg⁡max⁡p(x∣θ)×p(θ)\begin{aligned} \widehat\theta_{\rm{MAP}} &= \arg \max_\theta p(\theta \mid x)\\ &= \arg \max_\theta \frac{p(x \mid \theta) \times p(\theta)}{P(x)}\\ &= \arg \max p(x \mid \theta) \times p(\theta) \end{aligned} θMAP​​=argθmax​p(θ∣x)=argθmax​P(x)p(x∣θ)×p(θ)​=argmaxp(x∣θ)×p(θ)​

对其取对数：

arg⁡max⁡θp(x∣θ)⋅p(θ)=arg⁡max⁡θlog⁡∏i=0np(xi∣θ)p(θ)=arg⁡max⁡θ∑i=0nlog(p(xi∣θ)p(θ))=arg⁡max⁡θ∑i=0nlog(p(xi∣θ)+log(p(θ))\begin{aligned} \arg \max_θ p(x|θ)⋅p(θ) &= \arg \max_θ \log \prod_{i=0}^n p(x_i \mid θ)p(θ)\\ &= \arg \max_θ \sum_{i=0}^n log(p(x_i \mid θ)p(θ))\\ &= \arg \max_θ \sum_{i=0}^n log(p(x_i \mid θ)+log(p(θ)) \end{aligned} argθmax​p(x∣θ)⋅p(θ)​=argθmax​logi=0∏n​p(xi​∣θ)p(θ)=argθmax​i=0∑n​log(p(xi​∣θ)p(θ))=argθmax​i=0∑n​log(p(xi​∣θ)+log(p(θ))​

显然，随着数据量的增加，参数分布会更倾向于向数据靠拢，先验假设的影响会越来越小。

而结构风险最小化（structural risk minimization，SRM）：

min⁡f∈F1N∑i=1NL(yi,f(xi))+λJ(f)\min_{f\in\mathcal{F}} \frac{1}{N} \sum_{i=1}^N L(y_i,f(x_i))+\lambda J(f) f∈Fmin​N1​i=1∑N​L(yi​,f(xi​))+λJ(f)

其中 F\mathcal{F}F 为假设空间，可以看出当模型是条件概率分布、损失函数是对数损失函数、模型复杂度由模型的先验概率表示时，结构风险最小化就等价于最大后验概率估计。

#5：MLE与MAP的联系

MLE对参数 θ\thetaθ 的估计方法：

θ^MLE=arg⁡max⁡P(X;θ)=arg⁡max⁡P(x1;θ)P(x2;θ)…P(xn;θ)=arg⁡max⁡∏i=1nP(xi;θ)=arg⁡max⁡∑i=1nlog⁡P(xi;θ)=arg⁡min⁡−∑i=1nlog⁡P(xi;θ)\begin{aligned} \widehat\theta_{\rm{MLE}} &= \arg \max P(X;\theta)\\ &= \arg \max P(x_1;\theta)P(x_2;\theta) \ldots P(x_n;\theta)\\ &= \arg \max \prod_{i=1}^n P(x_i;\theta)\\ &= \arg \max \sum_{i=1}^n \log P(x_i;\theta)\\ &= \arg \min - \sum_{i=1}^n \log P(x_i;\theta) \end{aligned} θMLE​​=argmaxP(X;θ)=argmaxP(x1​;θ)P(x2​;θ)…P(xn​;θ)=argmaxi=1∏n​P(xi​;θ)=argmaxi=1∑n​logP(xi​;θ)=argmin−i=1∑n​logP(xi​;θ)​

MAP对参数 θ\thetaθ 的估计方法：

θ^MAP=arg⁡max⁡P(θ∣X)=arg⁡min⁡−log⁡P(θ∣X)=arg⁡min⁡−log⁡P(X∣θ)−log⁡P(θ)+log⁡P(X)=arg⁡min⁡−log⁡P(X∣θ)−log⁡P(θ)\begin{aligned} \widehat\theta_{\rm{MAP}} &= \arg \max P(\theta \mid X)\\ &= \arg \min -\log P(\theta \mid X)\\ &= \arg \min -\log P(X \mid \theta) - \log P(\theta) + \log P(X)\\ &= \arg \min -\log P(X \mid \theta) - \log P(\theta) \end{aligned} θMAP​​=argmaxP(θ∣X)=argmin−logP(θ∣X)=argmin−logP(X∣θ)−logP(θ)+logP(X)=argmin−logP(X∣θ)−logP(θ)​

所以MAP和MLE在优化时的不同就是在于增加了一个先验项：−log⁡P(θ)- \log P(\theta)−logP(θ)，因此可以大致给出它们之间的联系：

MAP(θ)≈MLE(θ)+P(θ)MAP(\theta) \approx MLE(\theta)+P(\theta) MAP(θ)≈MLE(θ)+P(θ)

当先验分布均匀之时，MAP 与 MLE 相等。直观讲，它表征了最有可能值的任何先验知识的匮乏。在这一情况中，所有权重分配到似然函数，因此当我们把先验与似然相乘，由此得到的后验极其类似于似然。因此，最大似然方法可被看作一种特殊的 MAP。

个人理解：若先验分布均匀，则 P(θ)P(\theta)P(θ) 为常数，则在MAP中

θ^MAP=arg⁡min⁡−log⁡P(X∣θ)−log⁡P(θ)=arg⁡min⁡−log⁡P(X∣θ)=θ^MLE\begin{aligned} \widehat\theta_{\rm{MAP}} &= \arg \min -\log P(X \mid \theta) - \log P(\theta)\\ &= \arg \min -\log P(X \mid \theta)\\ &= \widehat\theta_{\rm{MLE}} \end{aligned} θMAP​​=argmin−logP(X∣θ)−logP(θ)=argmin−logP(X∣θ)=θMLE​​

如果先验概率是均匀分布的，被称为无信息先验（non-informative prior），通俗的说就是“让数据自己说话”，此时贝叶斯方法等同于频率方法。随着数据的增加，先验的作用越来越弱，数据的作用越来越强，参数的分布会向着最大似然估计靠拢。而且可以证明，最大后验估计的结果是先验和最大似然估计的凸组合。

#6：共轭先验

如果先验分布和似然函数可以使得先验分布和后验分布有相同的形式，那么就称先验分布与似然函数是共轭的。它使得后验概率分布的函数形式与先验概率相同，因此使得贝叶斯分析得到了极大的简化。

例如，二项分布的参数之共轭先验就是Beta 分布，多项式分布的参数之共轭先验则是 Dirichlet 分布，而高斯分布的均值之共轭先验是另一个高斯分布。

对于给定的概率分布 P(θ∣x)P(\theta \mid x)P(θ∣x)，我们可以寻求一个与该似然函数 P(x∣θ)P(x \mid \theta)P(x∣θ) 共轭的先验分布 P(θ)P(\theta)P(θ)，如此一来后验分布 P(θ∣x)P(\theta \mid x)P(θ∣x) 就会同先验分布具有相同的函数形式。而且对于任何指数族成员来说，都存在一个共轭先验。

总结

经验风险最小化与结构风险最小化是对于损失函数而言的。可以说经验风险最小化只侧重训练数据集上的损失降到最低；而结构风险最小化是在经验风险最小化的基础上约束模型的复杂度，使其在训练数据集的损失降到最低的同时，模型不至于过于复杂，相当于在损失函数上增加了正则项，防止模型出现过拟合状态。这一点也符合奥卡姆剃刀原则：如无必要，勿增实体。

经验风险最小化可以看作是采用了极大似然的参数评估方法，更侧重从数据中学习模型的潜在参数，而且是只看重数据样本本身。这样在数据样本缺失的情况下，很容易管中窥豹，模型发生过拟合的状态；结构风险最小化采用了最大后验概率估计的思想来推测模型参数，不仅仅是依赖数据，还依靠模型参数的先验假设。这样在数据样本不是很充分的情况下，我们可以通过模型参数的先验假设，辅助以数据样本，做到尽可能的还原真实模型分布。
